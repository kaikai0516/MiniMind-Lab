# 🏋️ 训练流程

> 8 种训练模式，从预训练到 Agent 强化学习，循序渐进。

---

## 训练总览

```
第1步: 预训练       → 语言建模基础    [必须]
第2步: 全量 SFT     → 指令跟随能力    [必须]
第3步: 知识蒸馏     → 小模型学大模型   [可选]
第4步: LoRA 微调    → 特定领域适配    [可选]
第5步: DPO 偏好对齐 → 人类偏好优化    [可选]
第6步: GRPO 强化    → 群组相对策略优化 [可选]
第7步: PPO 强化     → 近端策略优化    [可选]
第8步: Agent RL     → 工具调用能力    [可选]
```

---

## 第1步: 预训练

```bash
cd trainer

# 最小复现 (~2h on RTX 3090 / RX 7900 XTX)
python train_pretrain.py \
    --data_path ../dataset/pretrain_t2t_mini.jsonl \
    --batch_size 32 \
    --learning_rate 5e-4 \
    --epochs 2

# 完整训练
python train_pretrain.py \
    --data_path ../dataset/pretrain_t2t.jsonl \
    --batch_size 64 \
    --epochs 3
```

> 输出: `../out/pretrain_768.pth`

---

## 第2步: 全量 SFT 微调

```bash
python train_full_sft.py \
    --data_path ../dataset/sft_t2t_mini.jsonl \
    --batch_size 32 \
    --learning_rate 1e-5 \
    --epochs 3
```

> 输出: `../out/full_sft_768.pth`

---

## 第3步: 知识蒸馏（可选）

学生模型（小）从教师模型（大/MoE）学习：

```bash
python train_distillation.py \
    --data_path ../dataset/sft_t2t_mini.jsonl \
    --student_hidden_size 768 --student_num_layers 8 \
    --teacher_hidden_size 768 --teacher_num_layers 8 \
    --teacher_use_moe 1 \
    --alpha 0.5 --temperature 1.5 \
    --batch_size 32 --epochs 6
```

| 参数 | 说明 |
|------|------|
| `--alpha` | 蒸馏损失权重 (0=只做CE, 1=只做蒸馏) |
| `--temperature` | 软化概率分布的温度 |

## 多卡训练 (DDP)

```bash
# N 卡并行 (NVIDIA / AMD 均支持)
torchrun --nproc_per_node N train_full_sft.py \
    --batch_size 32 --epochs 3

# 多机多卡
torchrun --nproc_per_node 4 --nnodes 2 \
    --node_rank 0 --master_addr 192.168.1.100 --master_port 29500 \
    train_pretrain.py
```

```bash
python train_pretrain.py --from_resume 1

```

---

## RL 参数速查

| 参数 | 默认 | 说明 |
|------|------|------|
| `--num_generations` | 6 (GRPO) / 4 (Agent) | 每 prompt 生成数 |
| `--beta` | 0.1 | KL 散度系数 |
| `--max_gen_len` | 1024 | 单次生成最大 token |
| `--loss_type` | `cispo` | `cispo` / `grpo` |
| `--epsilon` | 0.2 | clip epsilon |
| `--epsilon_high` | 5.0 | CISPO 上界 |
| `--thinking_ratio` | 0.9 (GRPO) / 0.1 (Agent) | `<think>` 启用概率 |
