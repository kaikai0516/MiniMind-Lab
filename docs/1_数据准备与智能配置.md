## 一、数据集下载

从 [ModelScope](https://www.modelscope.cn/datasets/gongjy/minimind_dataset/files) 下载：

```bash

    pretrain_t2t_mini.jsonl sft_t2t_mini.jsonl --local_dir ./dataset
```

## 二、DataLoaderHub — 多格式加载

**文件**: `dataset/data_loader.py`

自动识别文件格式，统一加载：

```python
from dataset.data_loader import DataLoaderHub

hub = DataLoaderHub()

# 自动解析格式
ds = hub.load("dataset/sft.csv")          # CSV
ds = hub.load("dataset/extra.parquet")    # Parquet
ds = hub.load("dataset/data.json")        # JSON

# 多文件合并
ds = hub.load_multi([
    "dataset/sft_a.jsonl",
    "dataset/sft_b.jsonl",
    "dataset/sft_c.csv"
], dataset_type="sft")

# 便利函数
from dataset.data_loader import load_pretrain_data, load_sft_data, load_dpo_data
ds = load_sft_data(["dataset/sft_*.jsonl"])
```

### 支持的格式

| 格式 | 检测方式 | 说明 |
|------|---------|------|
| `.jsonl` | 扩展名 | 标准 JSON Lines |
| `.json` | 扩展名 | JSON 数组或行 |
| `.csv` | 扩展名 | 逗号分隔 |
| `.parquet` | 扩展名 | 列存储 |
| `.txt` | 扩展名（降级） | 每行作为一条数据 |

### 列别名自动映射

| 规范列 | 自动识别别名 |
|--------|------------|
| `text` | `content`, `document`, `passage`, `sentence`, `body`, `article` |
| `conversations` | `messages`, `dialog`, `dialogue`, `chat`, `history` |
| `chosen` | `preferred`, `positive` |
| `rejected` | `dispreferred`, `negative` |

## 三、DataProcessor — 数据清洗

**文件**: `dataset/data_processor.py`

```python
from dataset.data_processor import (
    clean_text,       # Unicode 规范化 + 空白清理
    filter_by_length, # 按字符数过滤
    deduplicate,      # SimHash 近似去重
    describe_dataset, # 统计报告
    process_dataset,  # 一键全流程
)

# 一键清洗
clean = process_dataset(
    raw_dataset,
    min_length=10,
    max_length=2048,
    dedup_threshold=3,  # SimHash 汉明距离阈值
    clean=True,
    validate=True,
)
# 处理顺序: 清洗 → 验证 → 长度过滤 → 去重

# 查看统计
stats = describe_dataset(clean)
# → {total: 100000, avg_len: 385.2, p50: 280, ...}
```

## 四、auto_config — 智能训练分析

**文件**: `scripts/auto_config.py`

自动探测硬件 + 分析数据集 + 调用 AI = 输出最优训练参数。

```bash
# 基础用法
python scripts/auto_config.py \
    --data_path dataset/sft_t2t_mini.jsonl \
    --task full_sft \
    --api_key sk-xxx

# 输出
# → batch_size: 32, learning_rate: 1e-5, epochs: 3, ...
# → 可直接运行的训练命令

# Dry run: 只看 Prompt，不调用 API
python scripts/auto_config.py --data_path dataset/sft_t2t_mini.jsonl --dry_run
```

### 功能模块

| 模块 | 函数 | 检测内容 |
|------|------|---------|
| 硬件探测 | `hardware_probe()` | GPU 型号/显存 (CUDA/ROCm)、CPU、内存、磁盘 |
| 数据分析 | `analyze_dataset()` | 样本数、字段、长度/token 分布 |
| 显存估算 | `estimate_memory()` | 参数量、峰值显存 |
| AI 推荐 | `get_config_from_llm()` | 调 OpenAI 兼容 API 获取配置建议 |
