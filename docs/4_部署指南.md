# 🚀 部署指南

> 支持 NVIDIA CUDA · AMD ROCm · CPU 三种后端。

---

## 一、环境要求

- Python 3.10+
- 4GB+ RAM（训练推荐 16GB+）
- (可选) NVIDIA GPU + CUDA 12.x 或 AMD GPU + ROCm 6.x
- (可选) 8GB+ 显存（训练推荐 12GB+）

---

## 二、pip 安装

```bash
git clone https://github.com/kaikai0516/MiniMind-Lab
cd MiniMind-Lab

# 基础安装
pip install -e .

# 训练环境
pip install -e ".[train]"

# 全量安装
pip install -e ".[all]"
```

---

## 三、GPU 驱动安装

### NVIDIA CUDA

```bash
# CUDA 12.4 / 12.6
pip install torch==2.6.0 torchvision==0.21.0 --index-url https://download.pytorch.org/whl/cu124

# CUDA 12.1
pip install torch==2.5.1 torchvision==0.20.1 --index-url https://download.pytorch.org/whl/cu121

# CUDA 11.8
pip install torch==2.3.0 torchvision==0.18.0 --index-url https://download.pytorch.org/whl/cu118
```

验证:

```bash
python -c "import torch; print(torch.cuda.is_available(), torch.cuda.get_device_name(0))"
```

### AMD ROCm

适用: RX 7900 XTX, RX 7900 XT, RX 7800 XT, Instinct MI250X/MI300X

```bash
# ROCm 6.2
pip install torch==2.6.0 torchvision==0.21.0 --index-url https://download.pytorch.org/whl/rocm6.2

# ROCm 5.7
pip install torch==2.3.0 torchvision==0.18.0 --index-url https://download.pytorch.org/whl/rocm5.7
```

验证:

```bash
python -c "import torch; print(torch.cuda.is_available(), torch.cuda.get_device_name(0))"
```

> MiniMind-Lab 已做 ROCm 兼容：`trainer_utils.py` 的 cuDNN 调用有守卫，`auto_config.py` 能识别 AMD GPU。

### CPU

```bash
pip install torch==2.6.0 --index-url https://download.pytorch.org/whl/cpu
```

## 四、Docker 部署

```bash
# 构建
docker build -t minimind-lab:dev --target dev .

# 运行（挂载输出目录）
docker run --gpus all -p 8000:8000 -v $(pwd)/out:/app/out minimind-lab:dev

# API 服务
docker run --gpus all -p 8000:8000 minimind-lab:dev minimind-serve
```

基础镜像: `pytorch/pytorch:2.6.0-cuda12.4-cudnn9-runtime`

---

## 五、Makefile 快捷命令

```bash
# 安装
make install              # pip install -e .
make install-all          # pip install -e ".[all]"

# 训练 (通过 ARGS 传参)
make pretrain ARGS="--epochs 3 --batch_size 64"
make full-sft ARGS="--epochs 5"
make dpo    ARGS="--beta 0.15"
make grpo   ARGS="--num_generations 6"

# 推理
make eval       make serve     make chat      make auto-config

# 质量
make lint       make test      make clean     make format

# Docker
make docker-build             make docker-run
```

## 六、AMD GPU 注意事项

| 事项 | 状态 |
|------|------|
| `torch.cuda.*` API | ✅ 完全兼容 |
| bfloat16 / float16 | ✅ 支持 |
